Accordion/Expander directives example:

Premise: writing directives with Angualar is a very wide and complex subject. Don't expect to understand everything.
  • 'Expander' is a clickable control that shows/hides its content. It has a title and a content.
  • 'Accordion' holds a collection of expanderss, and when one is expanded, all the others in the collection are collapsed.
  • When created, 'Expander' looks for the accordion controller in the parent chain and register itself into it, so that accordion can keep a list of expanders. For such purpose, the accordion controller is passed as argument in the expander Link function.
  • When expander is clicked, it toggles itself and notifies the accordion controller so that he can collapse all the other expanders.
  • Both directives have their own separate controllers. Such controllers are available in the $scope object as field members (the "controller as" syntax, e.g. $scope.ctrl)
  • Controllers can access the $scope object because it can be injected.
